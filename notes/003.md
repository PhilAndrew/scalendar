## Timezone support

Scalendar will build a time based on the timezone you provide explicitly

    // Normal Object creation

    // You machine's timezone
    val now = Scalendar.now

    val tz = now.tz

    // UTC offset
    val offset = now.tz.offset
    
    // Display name of the TimeZone
    tz.name
    
    // TimeZone ID
    tz.id
